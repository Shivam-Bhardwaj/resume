# Job Description: SLAM Engineer

## About the Role
We are looking for a SLAM Engineer to join our autonomous robotics team. You will be responsible for developing and optimizing Simultaneous Localization and Mapping algorithms for our fleet of mobile robots.

## Responsibilities
- Design and implement real-time SLAM algorithms (e.g., VSLAM, Lidar-SLAM).
- optimize sensor fusion pipelines (IMU, Odometer, Lidar, Camera).
- Work with C++ and Python in a Linux environment.
- Deploy code to embedded edge devices (Jetson, x86).

## Requirements
- Strong proficiency in C++ and Python.
- Experience with ROS/ROS2 navigation stack.
- Hands-on experience with sensors like Velodyne Lidar, Realsense cameras.
- Knowledge of optimization libraries (g2o, CERES).
- Experience with Kalman Filters (EKF, UKF) and Graph-based SLAM.
